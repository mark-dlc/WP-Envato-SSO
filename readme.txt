This plugin is a OAuth2 Client plugin that provides Single Sign On capabilities with the Envato API

== Description ==

This plugin is designed and developed for use with the Envato OAuth API (build.envato.com). It's based on the Single Sing On Client plugin (http://wp-oauth.com)

Use Case;

You have a website targeted at Envato buyers, and you want them to be able to login.
Instead of having all your users create a new account on your website, you can simply let them sign-in using their Envato account.

== Installation ==

1. Install the Plugin in the Wordpress Plugin manager.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Single Sign On screen to configure the client

== Frequently Asked Questions ==

= Single Sign On Error =

In certain cases the user will be presented with a message "Single Sign On Error". This is because the use either has a
username already and the emails do not match or visa versa. The solution is to completely remove the associated user from
the client site.
